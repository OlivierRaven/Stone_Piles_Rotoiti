# Reinforcing kōura resilience using habitat enhancement structures

Olivier V. Raven | olivier.raven@icloud.com

**DOI:** To be added upon acceptance

## Overview

This repository contains all data, analysis code, and manuscript files for a before-after, control-impact (BACI) field study testing whether stone pile habitat enhancement structures provide effective habitat for kōura (*Paranephrops planifrons*) in Lake Rotoiti, Aotearoa New Zealand. Five stone pile sites were deployed along the shoreline, each paired with a control site and a naturally occurring rocky reference site, and monitored seasonally over 18 months for kōura and fish occurrence, abundance, and biomass.

## Reproducing the analysis

This project uses [Quarto](https://quarto.org/) and R.
All analyses are contained in `analysis.qmd`.

### Requirements

- R >= 4.4
- Quarto >= 1.4
- R packages: `MuMIn`, `lunar`, `glmmTMB`, `DHARMa`, `broom.mixed`,
  `emmeans`, `lmerTest`, `lme4`, `patchwork`, `tidyverse`, `writexl`, `readxl`

### Install R packages

```r
install.packages(c("MuMIn", "lunar", "glmmTMB", "DHARMa", "broom.mixed",
                   "emmeans", "lmerTest", "lme4", "patchwork",
                   "tidyverse", "writexl", "readxl"))
```

### Render the analysis

```bash
quarto render analysis.qmd
```

The rendered HTML analysis is available at:
<https://olivierraven.github.io/Stone_Piles_Rotoiti/>
